## 🔭 Featured Projects

### 📌 [Sentinel — Last-Mile Delivery Disruption Intelligence Platform](https://github.com/HarshithNR02/REPLACE_WITH_SENTINEL_REPO)
End-to-end platform on 6M+ real deliveries (Cainiao LaDe, 5 cities). Two models running side by side: a supervised LightGBM disruption classifier and an unsupervised Isolation Forest anomaly detector. Includes a Kafka streaming pipeline, a FastAPI serving layer with a feature store, a hybrid Text-to-SQL + FAISS RAG assistant, a simulated A/B test (two-proportion z-test), Evidently drift monitoring, and a live operational dashboard deployed on AWS EC2.

*Key finding: 30 couriers (0.6% of the network) generated 20% of all anomalies — turning 66K raw alerts into a 30-courier action list.*

### 📌 [KKBOX Churn Intelligence Platform](https://github.com/HarshithNR02/REPLACE_WITH_KKBOX_REPO) · [Live Demo](https://huggingface.co/spaces/harshith02/kkbox-churn-intelligence)
LightGBM churn model on 970K users × 110 features (time-based validation, AUC 0.9481). CLV-based segmentation into 5 actionable cohorts, deployed on Hugging Face Spaces with CI/CD.

### 📌 [Flight Delay Intelligence Platform](https://github.com/HarshithNR02/FLIGHT_DELAY_INTELLIGENCE_PLATFORM) · [Live Demo](https://harshith02-flight-delay-intelligence.hf.space)
LightGBM delay classifier on 18.2M US domestic flights with strict temporal validation (AUC 0.8629), SHAP analysis showing real_time_turn_gap as the #1 predictor, cascade/route-risk scoring, and a $2.78B annualized savings estimate. FastAPI + Docker + GitHub Actions CI/CD.

### 📌 [Walmart M5 Sales Forecasting](https://github.com/HarshithNR02/walmart_sales_forecasting)
LightGBM global model (WRMSSE 0.4656) across 30,490 item-store series, beating AutoARIMA and 7 other baselines. PostgreSQL business insights and a LangChain + GPT-4o-mini natural language query interface.
