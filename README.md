## 🔭 Featured Projects

### 📌 Sentinel — Last-Mile Delivery Disruption Intelligence Platform
End-to-end platform on 6M+ real deliveries (Cainiao LaDe, 5 cities). Two models running side by side: a supervised LightGBM disruption classifier and an unsupervised Isolation Forest anomaly detector. Includes a Kafka streaming pipeline, a FastAPI serving layer with a feature store, a hybrid Text-to-SQL + FAISS RAG assistant, a simulated A/B test (two-proportion z-test), Evidently drift monitoring, and a live operational dashboard.
*Key finding: 30 couriers (0.6% of the network) generated 20% of all anomalies — turning 66K raw alerts into a 30-courier action list.*
